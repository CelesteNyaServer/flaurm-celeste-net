A [Flarum](http://flarum.org) extension. celestenet

## Installation

Install with composer:

```sh
composer require smallmiao/flaurm-celeste-net:"*"
```

## Updating

```sh
composer update smallmiao/flaurm-celeste-net:"*"
php flarum migrate
php flarum cache:clear
```
